# Module - IO

Provides functions for input and output

## Function - Display

Prints the given values to the console

Symbol : io:display  
Parameters : Any number - Any type  
Parameters evaluated : Yes

Examples : 
- `(io:display 42)` => 42

## Function - Input

Reads a line from the console

Symbol : io:input  
Parameters : Any number - Any type  
Parameters evaluated : Yes

Examples : 
- `(io:input)` => 42
