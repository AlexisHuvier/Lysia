# Module - List

Provides functions for list operations

## Function - Create

Creates a new list

Symbol : list:create  
Parameters : Any number - Any type  
Parameters evaluated : Yes

Examples : 
- `(list:create 1 2 3)` => [1, 2, 3]

## Function - First

Returns the first element of a list

Symbol : list:first  
Parameters : 1 - list  
Parameters evaluated : Yes

Examples : 
- `(list:first [1 2 3])` => 1

## Function - Last

Returns the last element of a list

Symbol : list:last  
Parameters : 1 - list  
Parameters evaluated : Yes

Examples : 
- `(list:last [1 2 3])` => 3
